# ServiceHub

ServiceHub is a React-based home service booking application where users can book services like plumbing, electrical work, and AC repair. The project includes user login, admin management, booking status updates, and a responsive UI.

---

## Features

* User Login System
* Admin Login System
* Service Booking
* Customer Details Form
* Dashboard for Users
* Admin Panel
* Booking Status Update
* Pending / Completed Status
* Local Storage Data Management
* Responsive UI Design

---

## Technologies Used

* React JS
* React Router DOM
* JavaScript
* HTML
* CSS
* Vite

## Installation

### Clone Repository

```bash
git clone https://github.com/srimathik16-hue/servicehub.git
```

### Move to Project Folder

```bash
cd servicehub
```

### Install Dependencies

```bash
npm install
```

### Run Project

```bash
npm run dev
```
